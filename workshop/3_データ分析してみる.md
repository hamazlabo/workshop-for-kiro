# データ分析してみる

## このセクションの目的

Kiro CLI のコード開発以外の使い方を試してみて、利用方法のイメージを広げる。

## 3.1. 手順

### 3.1.1. データの内容を確認してもらう

```txt
次のデータの先頭を数十行読み込んで、どんなデータなのか教えて下さい。
/home/ubuntu/workshop-for-kiro/workshop/data/yokohama_data.csv
```

読み込んで、内容を教えてくれる。

```txt
これは**気象庁の気象観測データ（横浜地点の日別平均気温）**です。
```

### 3.1.2. データをグラフ化してもらう

```txt
このデータを折れ線グラフで表現して下さい。
```

回答例:

```txt
Streamlitアプリを起動しました。
横浜の1926年1月1日〜2026年6月13日（約100年分）の日別平均気温を折れ線グラフで表示しています。Streamlitのインタラクティブなグラフなので、ズームやホバーで値の確認もできます。
```

### 3.1.3. 様々な切り口で分析させてみる

前述のデータを更に細かく分析してみる。

```txt
横浜、大阪、札幌の平均気温を比較するグラフを作成して下さい。
```

グラフではなく、シンプルな分析をさせてみる。

```txt
ここ100年での平均気温の上昇率を調べて下さい。
```

そもそも、分析の切り口を聞きたい。

```txt
このデータを分析する際の切り口を教えて下さい。
```

### 3.1.4. 分析レポートを作成させる

データを分析して、レポートを作成してファイルとして保存する。

```txt
横浜、大阪、札幌のデータを分析して、レポートを作成して下さい。
テーマは100年間での温度上昇です。

- 季節ごとに、各都市の平均気温を比較したグラフと、データに基づいた考察
- 今後100年の見通し

の2項目でまとめて、pdf 形式で保存して下さい。
```
