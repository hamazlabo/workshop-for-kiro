# README

本ワークショップは、AI エージェントを活用した開発・運用を体験するものです。

どの順番から実施頂いても問題ありません。
但し、4 は、時間が余った向けのコンテンツですので、最後または難易度的に物足りない場合に挑戦してみて下さい。

## 事前作業: Kiro へのログイン

### 1. ターミナルを開く

1. Code Server 左上、オプションボタンを押下
2. 「Terminal」 > 「New Terminal」を押下

画面真ん中に、ターミナルが表示されれば完了です。

### 2. kiro へログイン

1. プロンプトに以下コマンドを入力
  > kiro-cli login --use-device-flow

  対話的にログインの案内が始まる。

2. **? Select login method ›** で、Use with Builder ID を選択し、Enter
3. **Open this URL:** に表示された URL をコピーし、ブラウザでアクセス
4. 予め作成した Builders ID でログイン
5. 「認証がリクエストされました」と表示された場合、「確認して続行」を押下
6. 「Kiro CLI がデータにアクセスすることを許可しますか？」と表示されるので、「アクセスを許可」を押下

「リクエストが承認されました」と表示され、ターミナルに「Logged in successfully」と表示されたら完了。

---

### 3. Kiro を停止する

ctrl + c を2回押下して、kiro を停止して下さい。

通常の usar名> のような表示になれば完了です。

---

以上で、事前作業は完了です。
